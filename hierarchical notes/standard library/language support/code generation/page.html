• Generation of Boilerplate Code
    • Copy & Move
      • shallow copy for relocatable types that don't need invalidation. (so that bitwise copy can be used)
      • deep copy for types consisting of deep copyable types.
      • move for types consisting of movable types.
      • move and deep copy from shallow copy for shallow types.
    • Comparison Operators
      • = from ≠ and vice versa
      • <,≤,>,≥ from = and either < or >
      • = from < and >
      • shallow equals for relocatable types.
      • deep equals for types consisting of types that support deep equals.
      • all comparison operators for types that should be lexicographically ordered and consist of types supporting at least < and =.
    • Constructors
      • no argument, initializing fields to the values that appear in the initializer of their declarations.
      • with as many parameters as there are fields without a specific attribute in the class declaration, initializing the fields using the corresponding arguments.
    • Deconstructors
      • with as many parameters as there are fields without a specific attribute in the class declaration, deconstructing the object to that many values.
    • Generators
      • (e.g. for property testing)
      • from constructors of immutable types iff generators exist for constructor parameter types.
      • from constructors and methods with side-effects iff generators exist for parameter types of constructors and methods.
    • Serialization & Deserialization
